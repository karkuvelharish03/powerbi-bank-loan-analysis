# Bank Loan Analysis Power BI Dashboard

## Project Overview

This project is an end-to-end Bank Loan Analysis Dashboard developed in Power BI to help financial institutions monitor lending performance, assess portfolio health, and analyze borrower behavior. The dashboard provides a comprehensive view of loan applications, funded amounts, repayments, interest rates, and debt-to-income ratios through interactive visualizations and KPI tracking.

## Dashboard Pages

### 1. Summary Dashboard

Provides a high-level overview of key lending metrics:

* Total Loan Applications: 38.6K
* Total Funded Amount: $435.8M
* Total Amount Received: $473.1M
* Average Interest Rate: 12.0%
* Average DTI Ratio: 13.3%
* Good Loan vs Bad Loan Analysis
* Loan Status Performance (Fully Paid, Current, Charged Off)

### 2. Overview Dashboard

Offers deeper insights into lending patterns:

* Monthly Loan Application Trends
* State-wise Loan Distribution
* Loan Term Analysis (36 vs 60 Months)
* Employment Length Analysis
* Loan Purpose Analysis
* Home Ownership Analysis

### 3. Details Dashboard

Provides transaction-level loan information including:

* Loan ID
* Loan Purpose
* Home Ownership
* Grade & Sub-Grade
* Issue Date
* Funded Amount
* Interest Rate
* Installment Amount
* Total Loan Amount

## Key Features

* Interactive KPI Cards
* Good Loan vs Bad Loan Performance Analysis
* Loan Status Monitoring
* Geographic Lending Analysis
* Borrower Segmentation
* Dynamic Filtering by Purpose, Grade, and State
* Drill-Through Reporting
* Responsive and User-Friendly Dashboard Design

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling (Star Schema)

## Business Value

This dashboard enables stakeholders to evaluate loan performance, identify risk trends, monitor repayment behavior, optimize lending strategies, and make informed business decisions through data-driven insights.
